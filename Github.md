## API
###### 发布:
```sh
https://api.github.com/repos/<用户>/<仓库>/releases
```
###### 目录:
```sh
https://api.github.com/repos/<用户>/<仓库>/contents/<目录>?ref=<分支>
```
###### 分支:
```sh
https://api.github.com/repos/<用户>/<仓库>/branches
```
